CompTIA ITF+
Infrastructure

CompTIA IT Fundamentals (ITF+) é uma introdução às habilidades de TI que ajuda os profissionais decidir se uma carreira em TI é certa para eles ou desenvolver uma compreensão mais ampla de TI.

Explore a ITF+

CompTIA A+
Core

CompTIA A + é o padrão da indústria para o lançamento de carreiras de TI e ensina o núcleo tecnologias de segurança a rede e virtualização.

Explore a A+

CompTIA Network+
Core

O CompTIA Network+ valida as habilidades técnicas necessárias para estabelecer, manter e solucionar problemas de redes essenciais com segurança.

Explore a Network+

CompTIA Security+
Core

CompTIA Security+ é uma certificação global que valida as habilidades de linha de base necessário para executar as principais funções de segurança.

Explore a Security+

CompTIA Data+
Core

O CompTIA Data+ ajuda você a iniciar sua carreira de análise de dados hoje. Aprenda sobre mineração de dados, manipulação de dados, análise de dados e muito mais.

Explore a Data+

CompTIA Project+
Core

O CompTIA Project+ oferece aos profissionais de TI os conceitos básicos para gerenciar com sucesso de pequeno a pequeno a projetos de médio porte.

Explore a Project+

CompTIA Cloud+
Core

CompTIA Cloud+ é uma certificação global que valida as habilidades necessárias para implantar e automatizar ambientes de nuvem seguros.

Explore a Cloud+
